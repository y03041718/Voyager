## 常见问题排查

### 问题1：内容宽度不对，左右留白太多
**原因：** 容器max-width限制
**解决：** 已在CSS中设置 `max-width: 100% !important`

### 问题2：字体太大或太小
**原因：** 字体单位使用px而不是pt
**解决：** 已在CSS中使用pt单位重新设置所有字体大小

### 问题3：图片超出页面
**原因：** 图片高度没有限制
**解决：** 已设置 `max-height: 60mm`

### 问题4：颜色不显示
**原因：** 没有勾选"背景图形"选项
**解决：** 在打印设置中勾选"背景图形"或"打印背景"

### 问题5：时间线位置不对
**原因：** 使用了相对单位（rem、em）
**解决：** 已改用绝对单位（mm）

### 问题6：内容被截断
**原因：** 没有设置分页控制
**解决：** 已添加 `page-break-inside: avoid`

### 问题7：间距太大或太小
**原因：** 使用了屏幕单位（px）
**解决：** 已改用打印单位（mm、pt）

## 优化建议

### 如果内容太多，一页放不下
1. 减少图片数量
2. 缩小图片尺寸
3. 减少文字描述
4. 调整字体大小

### 如果想要更紧凑的布局
在 `print.css` 中调整：
```css
@page {
  margin: 8mm 6mm; /* 减小边距 */
}

.mb-16, .mb-12 {
  margin-bottom: 4mm !important; /* 减小间距 */
}
```

### 如果想要更大的字体
在 `print.css` 中调整：
```css
body {
  font-size: 11pt !important; /* 增大基础字体 */
}

h1 {
  font-size: 22pt !important; /* 增大标题 */
}
```
